## 💡 FineDust

미세먼지 알리미 앱

<br><br>

## 🌍 지원 플랫폼

Android, iOS

<br><br>

## 🛠 사용

- Android Studio, Flutter
- Dart
- Clean Architecture, Riverpod(상태관리), Dio(HTTP 통신), Lottie(애니메이션), fl_chart(차트), geolocator(위치)

<br><br>

## ⭐️ 기능

- 실시간 대기질 조회
  - 사용자의 현재 위치 기반 측정소 탐색
  - PM10(미세먼지), PM2.5(초미세먼지) 실시간 조회
  - 아황산가스, 오존, 일산화질소 등 전체 항목 지원
- 상태(좋음/보통/나쁨/매우나쁨)에 따라 자동 애니메이션 변경
